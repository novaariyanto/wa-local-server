# Icon Instructions

1.  Gunakan **Opsi 1** (Online Converter) yang Anda pilih.
2.  Upload `logo.jpg` (yang ada di folder ini atau folder utama) ke converter:
    *   https://convertio.co/jpg-ico/
    *   https://www.icoconverter.com/
3.  Download hasilnya sebagai `icon.ico`.
4.  Simpan file tersebut di folder ini: `assets/icon.ico`.
